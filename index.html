    AJAX： var xhttp = null; 创建空对象， xhttp.setRequestHeader('Content-Type', 'application/x-www-form-urlencoded'); //使用post传输, if (window.XMLHttpRequest) { xhttp = new XMLHttpRequest(); //使用AJAX需先创建 } else { // code for IE6, IE5 xhttp = new ActiveXObject("Microsoft.XMLHTTP"); } open(method,url,async,user,psw) //规定一个请求 method: 请求类型 get 、post url: 文件位置（后端） async（可选）：true（默认）异步 false（同步） send() // 将请求发送 ！一个open对应一个send()选择上面 离的最近的open 若open内method属性为get 则发送的内容写在url 后用？连接 若open内method属性为post则所要传输的内容写 在send()括号里面 ps如果是对象的话需要进行转换 obj = JSON.stringify(obj) //转换为字符串 obj = JSON.parse(obj) //转换成对象 responseText // 获取已子字符串形式返回的数据 onreadystatechange //定义了当 readyState 属性发生改变时所 调用的函数。 readyState // 保存了XMLHttpRequest的状态 · 0：请求未初始化 。。请求为定义 · 1：服务器连接已建立。send发送 · 2：请求以已接收。 · 3：正在处理请求。 · 4：请求已完成且响应已就绪。 status //返回Http 状态 200 202.。。。。 statusText // 返回状态文本 200->OK 但要执行数据传递完后的动作需要添加判定 this.readyState == 4 && this.status == 200
